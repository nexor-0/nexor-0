## 🧠 Qui suis-je ?

Développeur web passionné, je vis et respire le code. J'aime creuser dans les fondamentaux du web — **HTML, CSS, JavaScript pur** — parce que comprendre les bases, c'est ce qui fait la différence entre quelqu'un qui utilise un framework et quelqu'un qui *comprend* vraiment ce qu'il se passe.

Chaque projet perso que je lance, c'est une nouvelle aventure : un problème à résoudre, une idée à concrétiser, un truc qui n'existait pas encore et que j'avais envie de voir exister.
